Javascript

  • for loop: allows a single code block to be executed over and over indicating a starting point and end point
    • Starting point of the for loop: a value for the loop to start at
    • Condition of the for loop: a condition for the loop to stop
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
